# Web Scraper Pro 🕷️

A powerful desktop GUI tool that extracts data from any website and exports it to Excel in one click. No coding required — just paste a URL and go.

Built with Python, PyQt5, BeautifulSoup, and openpyxl.

---

## Features

- 🔗 **Scrape Links** — extract all hyperlinks with text, URL, and type (internal/external)
- 📰 **Scrape Headings** — extract H1–H6 structure of any page
- 🖼️ **Scrape Images** — extract all image sources, alt text, and dimensions
- 📊 **Scrape Tables** — pull HTML tables directly into the results view
- 🎯 **Custom CSS Selector** — target any element on the page with precision
- 💾 **Export to Excel** — one-click export with formatted headers and auto-fit columns
- ⚡ **Background threading** — UI never freezes during scraping
- 🌑 **Dark themed UI** — clean PyQt5 interface

## How It Works

1. Run the script
2. Paste any URL into the input bar
3. Choose what to scrape: Links / Headings / Images / Tables / Custom Selector
4. Click **Scrape**
5. View results in the table
6. Click **Export to Excel** to save

---

## Requirements

```
Python 3.8+
PyQt5
requests
beautifulsoup4
openpyxl
```

Install all dependencies:

```bash
pip install PyQt5 requests beautifulsoup4 openpyxl
```

---

## Run

```bash
python web_scraper.py
```

## Use Cases

- Market research — scrape competitor product listings
- SEO auditing — extract all links and headings from a site
